# Whole-builder benchmark

Run the reproducible parse-to-codegen benchmark from the repository root:

```sh
benchmark/pipeline.rb --rules 200 --iterations 5 --seed 12345
```

The benchmark measures parse, normalize, LALR construction, and an explicit compact-table construction separately. The
`codegen_with_tables` stage measures the complete Ruby generator call, including the generator's own compact-table construction.
It also reports production count, state count, table cells, generated bytes, and a deterministic result digest. Use `--json`
for machine-readable output.

Wall-clock values are observational and intentionally have no pass/fail threshold. Identical iterations must produce the
same digest, so deterministic output regressions still fail immediately.
