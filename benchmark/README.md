# Whole-builder benchmark

Run the reproducible parse-to-codegen benchmark from the repository root:

```sh
benchmark/pipeline.rb --rules 200 --iterations 5 --seed 12345
```

The benchmark measures parse, normalize, LALR construction, compact-table construction, and Ruby code generation
separately. It also reports production count, state count, table cells, generated bytes, and a deterministic result digest.
Use `--json` for machine-readable output.

Wall-clock values are observational and intentionally have no pass/fail threshold. Identical iterations must produce the
same digest, so deterministic output regressions still fail immediately.
